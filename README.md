# Frontend Refresher Practice

This project is a from-scratch revision practice site built to reinforce the foundational concepts of HTML and CSS before returning to JavaScript frameworks.

It was created as a hands-on way to revisit core frontend skills such as:

- semantic HTML structure
- CSS layout and styling
- responsive design basics
- linking multiple pages and content sections
- building a simple, polished website from scratch

The goal was to strengthen confidence with the fundamentals so the next step could be learning JavaScript frameworks with a stronger base.

Live demo: https://taphexta.github.io/blog-frontend-practice/
